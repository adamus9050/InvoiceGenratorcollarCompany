The program is an online store created based on the needs of a company that produces shirt accessories. After placing an order, proceeding to the cart, and clicking the "Continue" button, the user can see the total cost of the order along with shipping details. At the same time, an invoice is generated and sent to the recipient's email. The invoice is in xlsx format and is fully supported by the Excel spreadsheet program.

Frameworks used:
-Entity Framework Core
-Asp Identity Core
-Open format XML
-Email Sender
